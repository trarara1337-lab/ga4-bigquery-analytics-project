# Google Analytics Data Analysis in BigQuery

## Overview

This project focuses on analyzing Google Analytics event data using BigQuery SQL.

The analysis included sessions, user behavior, event activity, and performance insights.

## Tasks

* Explore GA event tables
* Analyze user sessions
* Work with nested fields and arrays
* Use joins and window functions
* Build analytical insights

## Tools

BigQuery, SQL, Google Analytics, GA4

## Result

Prepared SQL queries for user behavior analysis, event tracking, and performance evaluation.

## Example Insights

* Ranked top users by total spend
* Identified first events in sessions
* Analyzed product revenue
* Worked with nested arrays and repeated fields
* Applied ranking and window functions
